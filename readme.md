# ww-components

[文档](https://storybook-template-ten.vercel.app/?path=/docs/configure--docs)


基于平时开发需求，积累的react通用组件。

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father`,

```bash
$ npm run build
```
